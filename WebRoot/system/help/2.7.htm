统计分析报表

统计分析报表包括以下几个功能模块：

n         按地点专业统计：按地点统计各公司盘点情况。

n         代维盘点监控：统计各代维公司盘点情况。

n         盘点结果报表：按部门统计盘点情况。

n         盘点明细报表：按部门统计具体盘点工单。

n         变动分析（按应用领域）：按应用领域统计资产变动情况。

n         变动分析（按资产大类）：按资产大类分析资产变动情况。

n         变动分析（按资产项）：按资产项分析资产变动。

n         盘点监控统计：按MIS资产统计各公司资产盘点情况。

n         盘点结果分析（省用）：按MIS资产统计各公司资产盘点排名。

n         盘点结果分析（市用）：按MIS资产统计各成本中心资产盘点排名。

n         资产变更分析：分析EAM资产与MIS资产变更情况。
